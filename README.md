# Copperleaf Café - aidekin live demo

A fictional café one-pager demonstrating the [aidekin](https://aidekin.com) embeddable AI
assistant on a real third-party origin: one script tag, a knowledge file built from the café's
own content, and everything (model, retrieval, chat) running in the visitor's browser.

- `index.html` - the café site with the embed snippet at the bottom
- `knowledge/copperleaf.md` - the café content the assistant is grounded in
- `copperleaf-knowledge.bin` - the prebuilt knowledge index the widget retrieves from
